# Code Drills

Code Drills is a small spaced-repetition coding practice app. It serves due coding tasks from a local JSON data store. Users copy the code through a three-level practice flow. Progress sent back to backend.

## Features

- Branch-based task categories
- Due-task filtering
- Three-level practice flow
- Progress saved after each completed task
- Review tasks prioritised by level

## Stack

- React + Vite frontend
- Express backend
- Local JSON file as the data store

## Run Locally

1. Start the backend
2. Start the frontend
3. Open the app in the browser

## Notes

- Tasks are currently stored in `server/src/data/learning.json`
- Only due tasks are shown
- Progress updates mutate the local JSON data
